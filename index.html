Computational Thinking

"The process of solving a real-world problem using a computer by expressing the inputs and outputs of the solution in a way that a computer can 'understand' it."

Algorithms

Algorithms help us describe how to solve a problem so that computers and humans can solve it. In this unit you will learn how describe an algorithm and why they are important in Computer Science.

Graphs

Graphs help us understand how things are connected to each other. Lots of real world problems can be simplified by using a graph, from chess problems to seating plans. In this block, you will learn how to turn real life problems into graphs and use algorithms on them to solve problems.

Complexity

Complexity determines the mathematical relationship between the size of the input and the amount of time or space an algorithm will use. In this block you will learn how to count the number of steps an algorithm takes, and use this to create a formula.

Recursion

Recursion is the process of calling the same function within a function. It allows us to solve problems that iterative methods can't solve. In this unit, you will learn the importance of recursive algorithms and how to read and create recursive algorihms.
